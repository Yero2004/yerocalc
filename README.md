# YeroCalc

A Python-based calculator application developed as part of learning and exploring GUI development and mathematical logic implementation.

## Purpose
This project was built to strengthen understanding of Python, event-driven programming, and user interface design while creating a functional and interactive application.

## Features
- Standard arithmetic operations
- Scientific mode with trigonometric and logarithmic functions
- Degree and radian mode switching
- Expression parsing and normalization for accurate calculations
- Custom animated UI buttons for improved user interaction

## Tech
- Python
- PySide6 (GUI framework)

## Notes
This project served as a hands-on learning experience focused on building real functionality, handling user input, and implementing mathematical operations within a graphical interface.

## Project Versions

This project has gone through multiple iterations:

- **v1.0 Yero_Calc(v1):**
  A basic calculator built using Tkinter, focusing on core functionality and simple UI design.

- **v2.0 Yero_Calc(v2):**
  A more advanced version built using PySide6, featuring:
  - Scientific mode
  - Degree/Radian switching
  - Improved UI design
  - Custom button animations

This progression reflects learning and improvement in GUI development, application structure, and overall design.
